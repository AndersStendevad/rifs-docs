The datasets available in rifs

LibriVoxDansk

This is a toy dataset for Danish ASR. This dataset consists of audio recordings from LibriVox with text from WeirdSpace Digital Library. All sound files and text files are in the public domain. As such there is no license with this dataset

Intended use

This dataset consists of danish fairy tales. It is unlikely that this dataset can be used for training. Though we consider it a good dataset for both developing methods, but also to test ASR systems.

WeirdSpace Digital Library

All text material is from WeirdSpace Digital Library. On the site they declare the following:

Public domain historier som pt. er tilgængelige. Vær venligst opmærksom på, at historierne ikke nødvendigvis er public domain i alle lande. link Last visited 22. feb 2023

Public domain

To the extent possible under law, Anders Stendevad has waived all copyright and related or neighboring rights to LibriVoxDansk. This work is published from: Danmark.

Den2Radio

Den2Radio is a Danish radio station that makes none commercial radio. It is driven by volunteers, and its radio programs are made my hobbyists. Some have worked in Broadcasting before. Others join to have their own radio program. This dataset is not public and therefore you may not use it for anything if you get access. All rights are reserved for Den2Radio and its organization.

Intended use

We expect this dataset to be used as a pre-traing corpus for Danish ASR. If the dataset gets labeled, then it is likely that it will be useful for fine-tuning as well.

Forskerzonen

This is a Danish ASR dataset that is not public. We have granted permission to use the speech and text data from Nana Elving Hansen at Videnskab.dk and the speaker of the dataset Lærke Sivkjær.

The data consists of articles made by researchers from Danish universities. There are 10 articles with different topics. The articles are read aloud by Lærke Sivkjær. The data is not aligned.

Intended use

We expect this dataset to be used as an evaluation dataset.

DanPASS

This is not the DanPASS dataset made by Nina Grønnum. Instead this is an adapter that can make the rifs-CLI work with the data without fuss. If you wish to use this dataset, then you first need to go to danpass.hum.ku.dk. Here you will need to make sure you understand what kind of data this is. Its limitations and bias.

If you decide that this dataset fits you purpose, then feel free to ask Nina Grønnum for access.

Intended use

We see this dataset as a small dataset for testing and maybe small scale training.

Preparing the data

First you need to get access and download the files. These need to be unzip’ed. Because the original data comes in a format that is not ready for training you have to go through some steps.

1. Filesturcture

You have to make sure you extract the files like this:


├── all.csv
├── audio
│   ├── dialogues_mono_soundfiles
│   │   └── *.wav
│   └── monologues
│       └── *.wav
├── .gitignore
├── README.md                                                                                                                                 ├── text
│   ├── dialogues_continuous.txt
│   ├── dialogues_mono_soundfiles
│   │   └── (Empty for now)
│   ├── monologues_continuous.txt
│   └── monologues
│       └── (Empty for now)
└── text_cleaning.py

2. run text_cleaning.py

dialogues_continuous.txt and monologues_continuous.txt contains the text found in the PDF from the DanPASS page. We have made a script to turn this into text that can be aligned by the rifs-CLI.

If you want to use the data remember to cite it in your paper.

NSTDanishSpråkbanken

This database was created by Nordic Language Technology for the development of automatic speech recognition and dictation in Danish.

This is not the NSTDanishSpråkbanken dataset. Rather it is an adapter to use the data in the rifs-CLI. To use this dataset you have to add all the .wav files to the dk folder.

Intended use

This can be used for both training and testing.

CommonVoiceDansk

This is an adapter that can make the rifs-CLI work with the CommonVoice data without fuss.

Common Voice is a publicly available voice dataset, powered by the voices of volunteer contributors around the world. People who want to build voice applications can use the dataset to train machine learning models.

At present, most voice datasets are owned by companies, which stifles innovation. Voice datasets also underrepresent: non-English speakers, people of colour, disabled people, women and LGBTQIA+ people. This means that voice-enabled technology doesn’t work at all for many languages, and where it does work, it may not perform equally well for everyone. We want to change that by mobilising people everywhere to share their voice.

The danish subset of CommonVoice is 12 hours of annotated speech. This is what we call CommonVoiceDansk

Intended use

This can be used for both training and testing.

DanskeTaler

This is a small dataset that is taken from Danske Taler. A danish initiative that has over 2500 Danish speeches on their site. We have tried to find a small subset for testing. The text is from Danske Taler. The audio is from various sources like Facebook and YouTube. All of these are public speeches.
