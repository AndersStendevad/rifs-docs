The CLI of the package

rifs

CLI for rifs package. Contains all the commands that the library supports. The CLI is written with click.

Usage: rifs [OPTIONS]

rifs [OPTIONS] COMMAND1 [ARGS]... [COMMAND2 [ARGS]...]...

Options

--version

Prints the version of the package.

--verbose

Enable verbose output.

--quiet

Disable all output.

--seed <seed>

Seed for the random number generator. Default: 0

--data-path <data_path>

Path to the data directory. Default: data

--model-path <model_path>

Path to the model directory. Default: models

--output-path <output_path>

Path to the output directory. Default: output

--noise-path <noise_path>

Path to the noise directory. Default: noise

--custom-dataset <custom_dataset>

Name of a custom dataset. Default: None

align

Usage: align [OPTIONS] DATASET

rifs align [OPTIONS] {LibriVoxDansk|Den2Radio|Forskerzonen|DanPASS|NSTDanishSp
           råkbanken|CommonVoiceDansk|DanskeTaler|Custom}

Options

--alignment-method <alignment_method>

Alignment method.

Options:

ctc | StateMachineForLevenshtein | StateMachineUnsupervised | StateMachineUnsupervisedNoModel

--model <model>

The path to the model to use for alignment. Can be a huggingface model or a local path.

--max-duration <max_duration>

Maximum duration of the audio files in seconds. Default: 10

Arguments

DATASET

Required argument

augment

Usage: augment [OPTIONS] DATASET

rifs augment [OPTIONS] {LibriVoxDansk|Den2Radio|Forskerzonen|DanPASS|NSTDanish
             Språkbanken|CommonVoiceDansk|DanskeTaler|Custom}

Options

--with-noise-pack <with_noise_pack>

Preprocess with noise pack. If not specified, no noise will be added.

--with-room-simulation

Preprocess with simulated room acoustics.

--with-speed-modification <with_speed_modification>

Preprocess with speed modification.

--include-audio-folder

Include the audio folder in data augmentation.

Arguments

DATASET

Required argument

merge-datasets

Usage: merge-datasets [OPTIONS] DATASET, … , DATASET NEW_DATASET

rifs merge-datasets [OPTIONS] [DATASET]... NEW_DATASET

Options

-s, --specify-dir <specify_dir>

Specify the directory to use for the dataset. If not set will copy everything. Useful to copy only, for example, the alignments. Default: None
