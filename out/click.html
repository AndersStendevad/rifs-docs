The CLI of the package

rifs

CLI for rifs package. Contains all the commands that the library supports. The CLI is written with click.

rifs [OPTIONS] COMMAND1 [ARGS]... [COMMAND2 [ARGS]...]...

Options

--version

Prints the version of the package.

--verbose

Enable verbose output.

--quiet

Disable all output.

--seed <seed>

Seed for the random number generator. Default: 0

--data-path <data_path>

Path to the data directory. Default: data

--model-path <model_path>

Path to the model directory. Default: models

--output-path <output_path>

Path to the output directory. Default: output

--noise-path <noise_path>

Path to the noise directory. Default: noise

align

Align DATASET

rifs align [OPTIONS] {LibriVoxDansk|Den2Radio}

Options

--alignment-method <alignment_method>

Alignment method.

Options:

ctc | StateMachineForLevenshtein | StateMachineUnsupervised | StateMachineUnsupervisedNoModel

--model <model>

The path to the model to use for alignment. Can be a huggingface model or a local path.

Arguments

DATASET

Required argument

augment

Augment DATASET

rifs augment [OPTIONS] {LibriVoxDansk|Den2Radio}

Options

--with-noise-pack <with_noise_pack>

Preprocess with noise pack. If not specified, no noise will be added.

--with-room-simulation

Preprocess with simulated room acoustics.

--with-voice-conversion

Preprocess with voice conversion.

Arguments

DATASET

Required argument

download-dataset

Download rifs DATASET

rifs download-dataset [OPTIONS] {LibriVoxDansk|Den2Radio}

Arguments

DATASET

Required argument

download-noise

Download NOISE-PACK from FreeSound.org

rifs download-noise [OPTIONS] NOISE_PACK

Arguments

NOISE_PACK

Required argument

finetune

Finetune model

rifs finetune [OPTIONS]

Options

--pretrained-model <pretrained_model>

Path to the pretrained model on disk or name of Huggingface model to base model on

--hours <hours>

Number of hours to train for. Default: 0

--minutes <minutes>

Number of minutes to train for. Default: 1

hubert-preprocess

Preprocess DATASET for hubert training

rifs hubert-preprocess [OPTIONS] {LibriVoxDansk|Den2Radio}

Options

--iteration <iteration>

Features for either first or second iteration of HUBERT training.

Options:

1st | 2nd

Arguments

DATASET

Required argument

pretrain

Pretrain model unsupervised

rifs pretrain [OPTIONS] {manifest_wav2vec2|wav2vec2_base|wav2vec2_large|wav2ve
              c2_conformer_base|wav2vec2_conformer_large|hubert_base|data2vec_
              base|data2vec2_base|data2vec2_large|speechT5_base}

Options

--fairseq-path <fairseq_path>

Path to the fairseq directory.

--manifest-source <manifest_source>

Optional: Path to wav files for manifest creation

Arguments

MODEL

Required argument
