rifsaugmentation.augmentation module

This module contains the augmentation classes.

The augmentation classes are used to augment the data. Right now we support the following augmentation classes:

  • NoiseAugmentation

  • RoomSimulationAugmentation

  • ModifySpeedAugmentation
