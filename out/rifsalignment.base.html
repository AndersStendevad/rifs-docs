Base class for all alignment algorithms.

abstract static align(audio_file: str, text_file: str, model: str, verbose: bool = False, quiet: bool = False, **kwargs) → List[TimedSegment]

Align the source and target audio files.

Parameters:

  • audio_file (str) – The path to the source audio wav file.

  • text_file (str) – The path to the source text file.

  • model (str) – The path to the model to use for alignment. Can be a huggingface model or a local path.

  • verbose (bool) – Whether to print the alignments progress with steps.

  • quiet (bool) – Prints nothing.
